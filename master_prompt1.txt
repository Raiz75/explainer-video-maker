Generate a long-form YouTube video script.

TRANSCRIPT:
[TRANSCRIPT]

GOAL:
You are given a transcript from a real video above.
Study its flow, structure, and progression of ideas.
Use it as your content blueprint — follow the same sequence of ideas and pacing,
but rewrite everything in your own words.

TONE — CRITICAL:
Do NOT write like an explainer or educator.
Write like a direct, confident narrator who is TELLING the viewer things:
- What happened
- How it works
- When it occurred
- Where it took place
- Why it matters
Use second person ("you") and present tense where natural.
Be direct, punchy, and authoritative — not academic, not preachy.
Examples of the right tone:
  BAD:  "In this segment, we will explore how the system works."
  GOOD: "Here's exactly how the system works."
  BAD:  "It is important to understand that..."
  GOOD: "This is what you need to know."

REQUIREMENTS:
- 8-10 segments total
- 1200-1800 words total
- Each segment must focus on ONE broad idea (follow the transcript's natural idea progression)
- Each segment is divided into microsegments
- Each microsegment = ONE sentence or short thought (10-30 words) mapped to ONE image and ONE pose

OUTPUT FORMAT (STRICT JSON):

Return ONLY a valid JSON array.

Each segment MUST include:
- segment number
- microsegments: array of objects, each with:
  - text: the narration line for that beat
  - image: a globally unique image number (integer)
  - pose: one of exactly these 5 values based on the tone of the line:
      "asking"      — the line poses a question or creates curiosity
      "authority"   — the line states a fact, conclusion, or authoritative claim
      "emphasis"    — the line stresses an important point or warning
      "explaining"  — the line describes a process, sequence, or how something works
      "pointingLeft"— the line references something shown in the image (directing attention)

FORMAT:

[
  {
    "segment": 1,
    "microsegments": [
      {"text": "Hook line here.", "image": 1, "pose": "asking"},
      {"text": "Next thought here.", "image": 2, "pose": "authority"},

  },
  {

    ]
  }
]

1. Image numbering MUST be GLOBAL
   - Start at 1
   - Increment by 1 across ALL segments and ALL microsegments
   - No duplicates, no gaps

2. Each microsegment gets exactly ONE image number

3. Each microsegment gets exactly ONE pose — choose based on the line's intent, not randomly

4. Each segment should have 3-6 microsegments

5. Each microsegment text: 10-30 words, one clear visual idea, direct storytelling tone

6. JSON must be valid:
   - no trailing commas
   - no extra text
